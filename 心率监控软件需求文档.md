# 华为手环心率监控软件需求文档

## 1. 项目概述

### 1.1 项目名称
Huawei HeartRate Monitor（华为心率监控悬浮窗）

### 1.2 项目背景
华为手环8系列支持蓝牙心率广播功能，开启后可作为标准BLE心率监测器向外广播心率数据。本项目旨在开发一款Windows桌面悬浮窗软件，实时接收并展示手环广播的心率数据，界面采用透明背景设计，仅显示线条、图标、图表等视觉元素。

### 1.3 项目目标
- 实时监控华为手环心率广播数据
- 提供美观的透明悬浮窗界面
- 支持心率历史趋势图表展示
- 低资源占用，稳定运行

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 蓝牙设备扫描与连接
- 自动扫描周围BLE设备
- 识别华为手环8系列设备（通过设备名称或广播特征）
- 支持手动选择设备连接
- 自动重连机制（断开后自动尝试重新连接）

#### 2.1.2 心率数据接收
- 使用标准蓝牙心率服务（GATT UUID: 0x180D）
- 解析心率测量特征值（UUID: 0x2A37）
- 支持心率值范围：0-255 BPM
- 数据更新频率：约1Hz（每秒更新一次）

#### 2.1.3 实时心率显示
- 当前心率数值大字体显示
- 心率状态图标（静息/运动/高强度）
- 连接状态指示器
- 电量显示（如果设备支持）

#### 2.1.4 心率历史图表
- 最近5分钟心率趋势折线图
- 心率区间分布（静息/燃脂/有氧/无氧）
- 平均心率、最高心率、最低心率统计
- 图表支持鼠标悬停查看具体数值

#### 2.1.5 心率报警功能
- 可设置心率上限报警阈值
- 可设置心率下限报警阈值
- 超出阈值时界面闪烁提示
- 可选声音报警

### 2.2 辅助功能

#### 2.2.1 窗口管理
- 窗口置顶显示（Always on Top）
- 窗口可拖动调整位置
- 窗口可调整大小
- 最小化到系统托盘
- 开机自启动选项

#### 2.2.2 界面自定义
- 透明度调节（50%-100%）
- 主题颜色切换（白色/黑色/自定义）
- 字体大小调节
- 图表显示/隐藏切换

#### 2.2.3 数据记录
- 心率数据本地存储（SQLite）
- 每日心率报告生成
- 数据导出（CSV格式）
- 历史数据查看

## 3. 非功能需求

### 3.1 性能需求
- CPU占用率：< 5%（正常运行时）
- 内存占用：< 100MB
- 数据延迟：< 2秒（从手环到屏幕显示）
- 启动时间：< 3秒

### 3.2 兼容性需求
- 操作系统：Windows 10 (build 19041+) / Windows 11
- 蓝牙版本：Bluetooth 4.0+ (BLE)
- 屏幕分辨率：支持1920x1080及以上
- .NET运行时：.NET 8.0 Desktop Runtime

### 3.3 安全性需求
- 本地数据加密存储
- 不收集用户隐私数据
- 不联网传输数据（纯本地应用）

### 3.4 可用性需求
- 界面简洁直观，无需学习成本
- 操作响应时间 < 100ms
- 提供中文/英文界面切换
- 错误提示清晰明确

## 7. 蓝牙协议说明

### 7.1 华为手环心率广播协议

华为手环8系列开启心率广播后，会作为标准BLE心率监测器广播数据：

| 服务/特征 | UUID | 说明 |
|-----------|------|------|
| Heart Rate Service | 0x180D | 心率服务 |
| Heart Rate Measurement | 0x2A37 | 心率测量值 |
| Body Sensor Location | 0x2A38 | 传感器位置 |
| Heart Rate Control Point | 0x2A39 | 控制点（可选） |

### 7.2 心率数据解析

Heart Rate Measurement (0x2A37) 数据格式：

```
Byte 0: Flags
  - Bit 0: Heart Rate Value Format (0=UINT8, 1=UINT16)
  - Bit 1: Sensor Contact Status
  - Bit 2: Sensor Contact Supported
  - Bit 3: Energy Expended Status
  - Bit 4: RR-Interval Status
  - Bit 5-7: Reserved

Byte 1: Heart Rate Value (UINT8 format)
  - Range: 0-255 BPM

Byte 1-2: Heart Rate Value (UINT16 format)
  - Range: 0-65535 BPM (实际不会超过255)
```

### 7.3 连接流程

```
1. 扫描BLE设备
   ↓
2. 过滤设备名称包含"HUAWEI"或"Huawei"
   ↓
3. 检查设备是否广播Heart Rate Service (0x180D)
   ↓
4. 建立GATT连接
   ↓
5. 发现服务和特征值
   ↓
6. 订阅Heart Rate Measurement通知
   ↓
7. 接收并解析心率数据
```
