# 心率监控软件界面设计文档

## 1. 设计概述

### 1.1 设计理念
- **极简主义**：只保留核心元素，去除所有多余装饰
- **透明悬浮**：背景完全透明，内容悬浮于桌面之上
- **信息层次**：通过大小、颜色、位置区分信息重要性
- **动态反馈**：使用动画提供实时反馈

### 1.2 设计规范

#### 字体规范
| 用途 | 字体 | 大小 | 粗细 |
|------|------|------|------|
| 心率数值 | DIN Alternate / Segoe UI | 48-72px | Bold |
| 统计标签 | Segoe UI | 12px | Regular |
| 统计数值 | Segoe UI | 14px | Semibold |
| 工具提示 | Segoe UI | 12px | Regular |

#### 颜色规范（暗色主题）
| 用途 | 颜色代码 | 说明 |
|------|----------|------|
| 心率数值 | #FFFFFF | 白色 |

public class HeartbeatAnimator
{

    private readonly DoubleAnimation _scaleYAnimation;

    {
        // 根据心率调整动画速度

        _scaleYAnimation.Duration = duration;

{
    public static readonly Color Resting = SKColors.Blue;        // 静息 < 100
    public static readonly Color FatBurn = SKColors.Green;       // 燃脂 100-120
    public static readonly Color Cardio = SKColors.Yellow;       // 有氧 120-140

            < 140 => Cardio,
            < 160 => Peak,
            _ => Maximum
        };
    }
}

## 5. 窗口拖动设计

### 5.1 拖动实现

```csharp
public partial class MainWindow : Window
{
    private bool _isDragging;
    private Point _dragStartPoint;
    
    public MainWindow()
    {
        InitializeComponent();
        
        // 允许窗口拖动
        MouseLeftButtonDown += OnMouseLeftButtonDown;
        MouseMove += OnMouseMove;
        MouseLeftButtonUp += OnMouseLeftButtonUp;
    }
    
    private void OnMouseLeftButtonDown(object sender, MouseButtonEventArgs e)
    {
        if (e.ClickCount == 1)
        {
            _isDragging = true;
            _dragStartPoint = e.GetPosition(this);
            CaptureMouse();
        }
    }
    
    private void OnMouseMove(object sender, MouseEventArgs e)
    {
        if (_isDragging)
        {
            var currentPoint = e.GetPosition(this);
            var offset = currentPoint - _dragStartPoint;
            
            Left += offset.X;
            Top += offset.Y;
        }
    }
    
    private void OnMouseLeftButtonUp(object sender, MouseButtonEventArgs e)
    {
        _isDragging = false;
        ReleaseMouseCapture();
        
        // 保存窗口位置
        SaveWindowPosition();
    }
}
```

## 6. 窗口吸附设计

### 6.1 屏幕边缘吸附

```csharp
public class WindowSnapper
{
    private const int SnapDistance = 20;
    
    public Point SnapToScreen(Point position, Size windowSize)
    {
        var screen = Screen.FromPoint(new System.Drawing.Point(
            (int)position.X, (int)position.Y));
        var workArea = screen.WorkingArea;
        
        var snappedX = position.X;
        var snappedY = position.Y;
        
        // 左边缘吸附
        if (Math.Abs(position.X - workArea.Left) < SnapDistance)
            snappedX = workArea.Left;
        
        // 右边缘吸附
        if (Math.Abs(position.X + windowSize.Width - workArea.Right) < SnapDistance)
            snappedX = workArea.Right - windowSize.Width;
        
        // 上边缘吸附
        if (Math.Abs(position.Y - workArea.Top) < SnapDistance)
            snappedY = workArea.Top;
        
        // 下边缘吸附
        if (Math.Abs(position.Y + windowSize.Height - workArea.Bottom) < SnapDistance)
            snappedY = workArea.Bottom - windowSize.Height;
        
        return new Point(snappedX, snappedY);
    }
}
```

public class ResponsiveLayout
{
    private readonly double _minWidth = 200;
    private readonly double _maxWidth = 400;
    
    public void AdjustLayout(double width)
    {
        if (width < 250)
        {
            // 紧凑模式：隐藏图表和统计
            ShowChart = false;
            ShowStats = false;
            HeartRateFontSize = 36;
        }
        else if (width < 300)
        {
            // 标准模式：显示图表，隐藏统计
            ShowChart = true;
            ShowStats = false;
            HeartRateFontSize = 48;
        }
        else
        {
            // 完整模式：显示所有
            ShowChart = true;
            ShowStats = true;
            HeartRateFontSize = 64;
        }
    }
}

    public static void AnimateNumber(TextBlock textBlock, int from, int to)
    {
        var animation = new DoubleAnimation
        {
            From = from,
            To = to,
            Duration = TimeSpan.FromMilliseconds(300),
            EasingFunction = new CubicEase { EasingMode = EasingMode.EaseOut }
        };
        
        animation.CurrentTimeInvalidated += (s, e) =>
        {
            var value = ((DoubleAnimation)s).GetCurrentValue(from, to);
            textBlock.Text = ((int)value).ToString();
        };
        
        textBlock.BeginAnimation(TextBlock.TagProperty, animation);
    }
}
