# 心率监控软件接口设计文档

## 1. 接口设计原则

### 1.1 设计原则
- **接口隔离**：每个接口只包含客户端需要的方法
- **依赖倒置**：高层模块依赖抽象接口，不依赖具体实现
- **单一职责**：每个接口只负责一个功能领域
- **可测试性**：接口设计便于单元测试和模拟

### 2.1 IBleService

```csharp
namespace HeartRateMonitor.Core.Interfaces
{
    /// <summary>
    /// 蓝牙低功耗服务接口
    /// </summary>
    public interface IBleService
    {
        /// <summary>
        /// 是否正在扫描
        /// </summary>
        bool IsScanning { get; }
        
        /// <summary>
        /// 当前连接状态
        /// </summary>
        ConnectionState State { get; }
        
        /// <summary>
        /// 当前连接的设备
        /// </summary>
        BleDevice ConnectedDevice { get; }
        
        /// <summary>
        /// 设备发现事件
        /// </summary>
        event EventHandler<BleDevice> DeviceDiscovered;
        
        /// <summary>
        /// 连接状态变化事件
        /// </summary>
        event EventHandler<ConnectionStateChangedEventArgs> ConnectionStateChanged;
        
        /// <summary>
        /// 心率数据接收事件
        /// </summary>
        event EventHandler<HeartRateData> HeartRateReceived;
        
        /// <summary>
        /// 开始扫描设备
        /// </summary>
        /// <param name="timeout">超时时间（秒）</param>
        /// <param name="cancellationToken">取消令牌</param>
        /// <returns>扫描任务</returns>
        Task StartScanAsync(int timeout = 30, CancellationToken cancellationToken = default);
        
        /// <summary>
        /// 停止扫描
        /// </summary>
        Task StopScanAsync();
        
        /// <summary>
        /// 连接到设备
        /// </summary>
        /// <param name="device">目标设备</param>
        /// <param name="cancellationToken">取消令牌</param>
        /// <returns>连接任务</returns>
        Task ConnectAsync(BleDevice device, CancellationToken cancellationToken = default);
        
        /// <summary>
        /// 断开连接
        /// </summary>
        Task DisconnectAsync();
        
        /// <summary>
        /// 获取已配对的设备列表
        /// </summary>
        /// <returns>设备列表</returns>
        Task<IEnumerable<BleDevice>> GetPairedDevicesAsync();
    }
    
    /// <summary>
    /// 连接状态变化事件参数
    /// </summary>
    public class ConnectionStateChangedEventArgs : EventArgs
    {
        public ConnectionState OldState { get; set; }
        public ConnectionState NewState { get; set; }
        public string DeviceId { get; set; }
        public string ErrorMessage { get; set; }
    }
}
```
